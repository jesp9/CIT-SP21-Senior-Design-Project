This project will introduce our audience with a website that allows you to check live updates on Covid-19 cases and Covid-19 vaccination cases. We will be pulling an API and using a lamba function to automate a scheduler in order to update our website daily at a specific time. To display our data, we will be using graphs and heat maps of the country so it is visually appealing and helps users understand the importance of this pandemic. This way, any user on our website can be quickly notified with how we are progressing as a nation towards overcoming this historical pandemic.
